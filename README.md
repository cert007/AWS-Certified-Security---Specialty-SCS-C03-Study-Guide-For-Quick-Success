## SCS-C03 Exam Content Domains and Weightings

The updated SCS-C03 exam consists of **six clearly defined domains**, reflecting modern cloud security responsibilities:

### Domain Breakdown

- **Domain 1: Detection** – 16%
    
    Focuses on monitoring, logging, threat detection, and security visibility.
    
- **Domain 2: Incident Response** – 14%
    
    Covers investigation, containment, remediation, and post-incident activities.
    
- **Domain 3: Infrastructure Security** – 18%
    
    Examines network security, compute protection, and secure architecture design.
    
- **Domain 4: Identity and Access Management (IAM)** – 20%
    
    Emphasizes authentication, authorization, least privilege, and identity governance.
    
- **Domain 5: Data Protection** – 18%
    
    Includes encryption, key management, data classification, and secure storage.
    
- **Domain 6: Security Foundations and Governance** – 14%
    
    Covers compliance, risk management, shared responsibility, and security best practices.
